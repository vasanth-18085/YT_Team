## PRE-LAUNCH CHECKLIST

### Financial & Time Commitment

- [ ] 3-5 year financial runway secured
- [ ] Stable income source established
- [ ] 15-20 hours per week available consistently
- [ ] Family/life commitments discussed and approved

### Success Definition

- [ ] Dream day written (hour by hour)
- [ ] Dream week written (activities detailed)
- [ ] Dream year written (doors opened described)
- [ ] Success metrics clearly defined (money/time/experiences/freedom)
- [ ] "Why YouTube?" answer written in journal

### Channel Foundation

- [ ] Channel positioning chosen (Teacher/Hero/Performer/Commentator)
- [ ] Niche selected (genuine interest confirmed)
- [ ] Unique angle/approach defined clearly
- [ ] Target audience demographics identified
- [ ] Value proposition written (one paragraph)
- [ ] Brand colors selected (set of 4)
- [ ] Font style chosen (consistency)
- [ ] Channel name decided
- [ ] Channel banner created
- [ ] Profile picture designed
- [ ] About section written

### Niche Research (40-60 Hours)

- [ ] Top 20 channels in niche identified
- [ ] 30-50 thumbnails screenshot and analyzed
- [ ] Top 20 video hooks transcribed
- [ ] 100+ comments read and documented
- [ ] Reddit threads researched
- [ ] Quora questions analyzed
- [ ] Research deck created (Google Slides)
- [ ] Thumbnail patterns documented
- [ ] Title formulas identified
- [ ] Content gaps discovered
- [ ] Common formats noted
- [ ] Brand sponsors identified
- [ ] CPM research completed
- [ ] TAM (Total Addressable Market) calculated

### Idea Bank

- [ ] 100+ video title ideas generated
- [ ] Ideas organized by category (FIIRE method)
- [ ] Top 30 ideas identified
- [ ] Development tier (40 ideas) created
- [ ] Archive tier (30 ideas) stored
- [ ] SEAS scoring completed for top 30
- [ ] Top 10 ideas validated with peers/testing

### Equipment & Tools

- [ ] Camera ready (iPhone or existing camera)
- [ ] Microphone functional (phone mic acceptable)
- [ ] Lighting setup arranged (window/basic)
- [ ] Editing software installed (iMovie/DaVinci Resolve)
- [ ] Thumbnail tool ready (Canva account)
- [ ] Google Docs for scripts
- [ ] Google Sheets for tracking
- [ ] Google Slides for research deck

### Systems Setup

- [ ] Tracking spreadsheet created (CTR, Retention, AVD)
- [ ] Production board system established
- [ ] Idea management system ready
- [ ] Weekly workflow calendar set
- [ ] Analytics review schedule planned

### Support System

- [ ] YouTube creator group identified (5-10 people)
- [ ] Or AI feedback group created (5 personalities)
- [ ] Weekly accountability check-in scheduled
- [ ] Peer feedback system established

### Mental Preparation

- [ ] 3 critical commitment questions answered honestly
- [ ] Super Mario Effect mindset adopted
- [ ] Expectation of 100+ videos before success accepted
- [ ] Financial pressure removed from creative process
- [ ] Long-term vision (3-5 years) embraced

## WEEKLY WORKFLOW CHECKLIST

### Monday: Review & Plan (1-2 hours)

- [ ] Review last week's video performance (all 3 metrics)
- [ ] Document learnings in tracking spreadsheet
- [ ] Note specific retention drop-off points
- [ ] Identify patterns across recent videos
- [ ] Select next video concept from top-tier idea bank
- [ ] Research trending topics in niche (VidIQ/manual)
- [ ] Check competitor uploads from past week

### Tuesday: Pre-Production (3-4 hours)

- [ ] Write full script in Google Doc
- [ ] Include title at top in bold
- [ ] Thumbnail reference added
- [ ] Hook section written (first 30 seconds)
- [ ] R.A.I.N.Y framework applied to intro
- [ ] Structure outlined (beginning/middle/end)
- [ ] Unanswered questions maintained throughout
- [ ] Design 2-3 thumbnail variants
- [ ] Refine first 30 seconds (minimum 5 iterations)
- [ ] Apply 3 B's (Bold, Basic, Brief) to entire script
- [ ] Validate SEAS criteria met for idea
- [ ] Get peer feedback if possible (share script)

### Wednesday: Production (2-4 hours)

- [ ] Review script one final time
- [ ] Set up recording space (lighting, camera, audio)
- [ ] Test equipment before recording
- [ ] Film hook (record multiple takes - at least 5)
- [ ] Film main content following script
- [ ] Record B-roll if needed
- [ ] Keep setup simple and sustainable (52-week test)
- [ ] Aim for 85% completion mindset
- [ ] Save all raw files properly organized

### Thursday: Post-Production (3-5 hours)

- [ ] Import footage to editing software
- [ ] Select best hook take
- [ ] Edit video with auto-cuts (remove dead air)
- [ ] Add auto-zooms for visual interest
- [ ] Enhance audio (normalize, remove background noise)
- [ ] Add B-roll and pattern interrupts
- [ ] Ensure focal point clarity throughout
- [ ] Apply 85% rule (don't over-edit)
- [ ] Create 2-3 related shorts from footage
- [ ] Export in recommended YouTube settings

### Friday: Finalize & Publish (2-3 hours)

- [ ] Final video review (is it 85% done? If yes, ship it!)
- [ ] Optimize title (40-60 characters, clear, persuasive)
- [ ] Write description (first 2 lines most important)
- [ ] Add relevant tags (10-15 tags)
- [ ] Select best thumbnail variant (A/B test if possible)
- [ ] Add end screen elements
- [ ] Add cards at appropriate moments
- [ ] Enable monetization settings
- [ ] Schedule or publish video
- [ ] Share on social media
- [ ] Post related short to drive traffic
- [ ] Update production board status

### Saturday-Sunday: Ideation & Growth (2-3 hours)

- [ ] Generate 10+ new video ideas
- [ ] Add to idea bank with initial notes
- [ ] Watch 5+ competitor videos in niche
- [ ] Take notes on successful elements
- [ ] Analyze top comments for content gaps
- [ ] Respond to comments on your videos (first 24-48 hours critical)
- [ ] Engage with community meaningfully
- [ ] Research next week's topic deeply
- [ ] Read relevant Reddit threads
- [ ] Check Quora for questions
- [ ] Reach out to 3 creators (connection building)
- [ ] Update idea bank organization
- [ ] Review and organize thumbnail/title inspiration

**Total Weekly Time: 15-20 hours**

## PER-VIDEO PRODUCTION CHECKLIST

### Pre-Production Phase

#### Idea Validation

- [ ] SEAS score calculated (minimum threshold met)
- [ ] Napkin test passed (visual anchors identified)
- [ ] CCN Fit verified (works for core, casual, new viewers)
- [ ] One-page development completed
- [ ] Peer feedback received and considered
- [ ] Test conducted (social media/short/conversation)

#### Script Development

- [ ] Title written at top in bold (40-60 characters)
- [ ] Thumbnail reference included
- [ ] Hook section written (30 seconds maximum)
    - [ ] Validates the click
    - [ ] Opens new question
    - [ ] Includes surprise element
- [ ] R.A.I.N.Y framework applied:
    - [ ] R - Result stated clearly
    - [ ] A - Objection addressed with proof
    - [ ] I - Instant (under 40 seconds total)
    - [ ] N - Why Now (urgency/relevance)
    - [ ] Y - Why You (authority established)
- [ ] Structure outlined:
    - [ ] Beginning (setup + unanswered question)
    - [ ] Middle (value delivery + new questions)
    - [ ] End (final answer + CTA)
- [ ] Unanswered questions maintained throughout
- [ ] 3 B's applied to all text:
    - [ ] Be Bold (authority words, no hedging)
    - [ ] Be Basic (simple language, minimal jargon)
    - [ ] Be Brief (under 25 words/sentence, under 3 sentences/paragraph)
- [ ] Audience language incorporated
- [ ] Hook refined 5+ times
- [ ] Target length determined (based on positioning)
- [ ] Script read aloud for flow

#### Title Creation

- [ ] 10-20 variations generated
- [ ] AI used for additional variations
- [ ] Character count: 40-60 characters
- [ ] Clarity rating by third party: 3+/5
- [ ] Persuasive? Yes
- [ ] Attention factor included:
    - [ ] Definitive/Confidence OR
    - [ ] Question/Curiosity OR
    - [ ] Wow Factor/Shocking
- [ ] Engaging language (niche power words)
- [ ] YouTube group consensus obtained
- [ ] Compared against competitor titles
- [ ] Final title selected and documented

#### Thumbnail Design

- [ ] 2-3 distinct variations created
- [ ] 5C Framework verified:
    - [ ] Composition (framing, angles, placement)
    - [ ] Context (passes 18% rule, glance test)
    - [ ] Clean (3 or fewer elements)
    - [ ] Curiosity (hook/question/contradiction)
    - [ ] Color (brand colors, theory applied)
- [ ] Unthumbnail approach used:
    - [ ] Original raw image (emotional)
    - [ ] Enhanced version
    - [ ] 2-4 words text maximum
    - [ ] Familiar symbols (arrows/emojis sparingly)
- [ ] Glance test passed (mobile quick scroll)
- [ ] Works without title (independent understanding)
- [ ] Works with title (complete package)
- [ ] Font consistent with brand
- [ ] High resolution (1280x720 minimum)
- [ ] YouTube group feedback received
- [ ] Best variant selected

### Production Phase

#### Setup

- [ ] Script printed or on second screen
- [ ] Recording space prepared
- [ ] Camera positioned correctly
- [ ] Lighting tested (natural or artificial)
- [ ] Audio levels tested
- [ ] Background appropriate for content
- [ ] Equipment charged/ready

#### Filming

- [ ] Multiple hook takes recorded (minimum 5)
- [ ] Best hook delivery selected
- [ ] Main content filmed following script
- [ ] Allow for natural deviations (keep energy)
- [ ] B-roll captured if needed
- [ ] Visual anchors demonstrated
- [ ] Energy maintained throughout
- [ ] Process kept lean (sustainable for 52 weeks)
- [ ] 85% mindset maintained (not perfection)
- [ ] All footage backed up

### Post-Production Phase

#### Editing

- [ ] Best hook take selected (90%+ retention goal)
- [ ] Dead air removed (auto-cuts)
- [ ] Visual interest maintained (auto-zooms)
- [ ] Audio enhanced (normalized, cleaned)
- [ ] B-roll inserted appropriately
- [ ] Pattern interrupts added
- [ ] Focal point clear throughout
- [ ] Hook-delivery cycle implemented
- [ ] Multiple hooks throughout video
- [ ] Unanswered questions maintained
- [ ] Experience focused (welcome/belonging)
- [ ] Not over-edited (focal point editing)
- [ ] 85% rule applied (ship when ready)

#### Shorts Creation

- [ ] 2-3 shorts extracted from long-form
- [ ] Each short tests specific angle
- [ ] Shorts optimized for vertical format
- [ ] Hooks adapted for short-form
- [ ] Related to main video (drive traffic)

### Upload Phase

#### Metadata Optimization

- [ ] Final title entered (40-60 characters)
- [ ] Description written:
    - [ ] First 2 lines compelling (visible before "show more")
    - [ ] Timestamps added
    - [ ] Links included (social, resources)
    - [ ] Keywords naturally incorporated
- [ ] 10-15 relevant tags added
- [ ] Thumbnail uploaded (best variant)
- [ ] End screen added (subscribe + video suggestions)
- [ ] Cards added at appropriate moments
- [ ] Playlist selected
- [ ] Category selected
- [ ] Comments enabled
- [ ] Age restrictions checked

#### Pre-Publish Check

- [ ] Video rendered correctly
- [ ] Audio synced properly
- [ ] Thumbnail displays correctly
- [ ] Title and thumbnail work together
- [ ] Description has no typos
- [ ] All links functional
- [ ] Premiere or schedule time set
- [ ] Reminder set to engage with comments

#### Post-Publish Actions (First 24-48 Hours)

- [ ] Pinned comment posted (question to boost engagement)
- [ ] Respond to first 10 comments within 1 hour
- [ ] Share on relevant platforms (Twitter, LinkedIn, etc.)
- [ ] Post related short to YouTube
- [ ] Monitor first hour performance
- [ ] Check if stuck in "review" or "limited"
- [ ] Track CTR at Hour 1, Day 1, Day 7, Day 30
