## SERIES TITLE SYSTEM

The Parva Narration series uses a **three-part naming system**:

```
Mahabharata — [Parva Name] | Episode [N]: [Key Event]
```

**Why this structure:**
- `Mahabharata —` → Series brand prefix (searchable, consistent, algorithm-learnable)
- `[Parva Name]` → Parva-level keyword territory (completely unoccupied, see competitive analysis)
- `| Episode [N]` → Signals episodic structure → return viewer behavior
- `: [Key Event]` → Curiosity hook within the episode

**Character count target:** 45–60 characters for the full title including prefix.

## EPISODE TEMPLATE FOR YEAR 3+ (Unassigned V-Numbers)

Use this template for every future Parva Narration episode not yet assigned a V-number:

```
Title:   Mahabharata — [Parva Name] | Episode [N]: [Key Event]
Length:  45–60 characters
Backup:  Mahabharata — [Parva Name] | Ep [N]: [Key Event Alternative]

Top Pick criteria:
1. "Mahabharata —" prefix always present (series brand)
2. Full parva name OR "Ep" abbreviation if parva name is long (Anushasana, Ashvamedhika)
3. Episode number always present
4. Key event in under 25 characters
5. Total: 45–60 characters, ✅

Long-parva title adjustment:
- Anushasana Parva → "Mahabharata — Anushasana Parva | Ep N: [Event]" = 52–58 chars ✅
- Ashvamedhika Parva → "Mahabharata — Ashvamedhika | Ep N: [Event]" = drop "Parva" if over 60
- Mausala Parva → short parva name, full title stays under 60 chars easily
```
